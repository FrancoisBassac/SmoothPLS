# Contributing to SmoothPLS

Hello! Thanks for checking out the project. 

## Current Status
I am actively developing this package as part of my PhD research. To keep the development focused and consistent with my thesis goals:

**Please contact me first before opening a Pull Request or proposing major changes if we are still below version 0.2.0.** I'd be more than happy to discuss ideas, bugs, or potential collaborations via GitHub Issues or email!

## How to contribute
1. Feel free to open an **Issue** for bugs or feature requests.
2. If you want to help, let's talk first so we don't double the work!

Thanks for your support!
